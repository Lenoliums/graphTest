# graphTest
Приложение для отображения графа

## Подключение и настройка

_Установка зависисмостей_:

```shell
yarn install
```

yarn@1.22.22

_Для запуска приложения в корневой папке выполнить команду_:

```shell
yarn start  
```

_Для запуска тестов в корневой папке выполнить команду_

```shell
yarn test
```
